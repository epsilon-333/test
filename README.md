# 반응속도 & 클릭속도 테스트

로컬에서 간단히 사용할 수 있는 테스트 페이지입니다.

사용법
- `index.html` 파일을 브라우저로 열어 사용하세요.
- 반응속도 테스트: 1,3,5,10 중 시도 횟수를 선택하고 시작. 빨간원이 초록색이 되면 클릭하세요. 끝나면 꺾은선 그래프와 최고/최저/평균을 표시합니다.
- 클릭속도 테스트: 3,5,10초 중 시간을 선택하고 시작. 큰 영역을 가능한 많이 클릭하세요. 끝나면 총 클릭수를 보여줍니다.

파일
- `index.html` — 메인 페이지
- `styles.css` — 스타일
- `script.js` — 동작 로직 및 차트

의견이나 개선할 점이 있으면 알려주세요.
